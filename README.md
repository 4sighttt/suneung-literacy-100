# suneung-literacy-100

수능 국어 비문학(독서) 영역을 위한 **문장 기반 문해력 훈련 시스템**

## 이 프로젝트는 무엇인가

수능 국어에서 반복 출현하는 **189개 핵심 개념어**를 단어 암기가 아닌 **문장 해석 훈련**으로 학습한다.

- 단어를 외우는 것이 아니라, 문장 속에서 개념이 어떻게 작동하는지 이해한다.
- 한 문장에 여러 개념이 동시에 등장하여, 실제 지문과 유사한 사고를 훈련한다.
- 약 50~70개 문장으로 189개 어휘를 커버한다.

## 학습 단위 구조

각 문장은 동일한 구조로 구성된다:

```
1. 문장        — 핵심 개념이 자연스럽게 포함된 문장
2. 문장 분해    — 전제 / 주장 / 조건 / 결론 등으로 구조 분리
3. 핵심 개념어  — 각 단어의 의미를 문장 맥락에 맞게 설명
4. 구조 요약    — 문장의 논리 구조를 한 줄로 정리
5. 포함 범주    — 해당 개념어가 속하는 분류 태그
```

## 학습 방법

```
1단계: 문장을 읽는다
2단계: 문장 분해를 보며 논리 구조를 파악한다
3단계: 핵심 개념어의 의미를 문장 맥락에서 이해한다
4단계: 구조 요약을 참고하여 문장을 다시 해석한다
```

## 예시

> 귀납적 추론은 개별 사례에서 일반화된 결론을 도출하지만, 단 하나의 반례만으로도 그 타당성이 흔들릴 수 있다.

**문장 분해**
- 전제: 귀납적 추론은 개별 사례 → 일반 결론 구조다
- 주장: 반례 하나가 타당성을 위협할 수 있다

**핵심 개념어**

| 개념어 | 이 문장에서의 의미 |
|--------|---------------------|
| 귀납 | 개별 관찰로부터 일반 법칙을 이끌어내는 추론 방식 |
| 일반화 | 제한된 사례를 전체에 적용하는 것 |
| 반례 | 일반화된 주장을 무너뜨리는 반대 사례 |
| 타당성 | 추론 과정이 논리적으로 성립하는 정도 |

**구조 요약:** 귀납의 작동 원리를 제시한 뒤, 그 한계를 반례로 지적하는 구조

**포함 범주:** `논리·추론`
