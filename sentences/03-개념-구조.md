## 문장 01

> 어떤 대상의 본질이란 그것이 그것일 수 있게 하는 핵심 속성을 가리키며, 부수적 속성이 변하더라도 본질이 유지되는 한 그 대상의 정체성은 보존된다.

### 문장 분해
- 정의: 본질 = 대상을 그것이게 하는 핵심 속성
- 주장: 부수적 속성이 변해도 본질이 유지되면 정체성은 보존된다

### 핵심 개념어

| 개념어 | 이 문장에서의 의미 |
|--------|---------------------|
| 본질 | 대상이 그 대상으로 존재하기 위해 반드시 갖추어야 하는 핵심적 성질 |
| 속성 | 대상이 지니고 있는 성질이나 특징 |

### 구조 요약
본질과 부수적 속성을 구분하고, 정체성 보존의 조건을 제시하는 구조

### 포함 범주
`개념·구조`

## 문장 02

> 개념은 여러 대상에서 공통된 속성을 추출하여 형성되며, 하나의 개념에 대해 정의를 내린다는 것은 그 개념의 의미 범위와 적용 기준을 확정하는 작업이다.

### 문장 분해
- 전제: 개념은 공통 속성 추출로 형성된다
- 결론: 정의 = 의미 범위와 적용 기준을 확정하는 것

### 핵심 개념어

| 개념어 | 이 문장에서의 의미 |
|--------|---------------------|
| 개념 | 여러 대상의 공통 속성을 묶어 형성한 추상적 관념 |
| 정의 | 개념의 의미와 적용 범위를 명확히 규정하는 행위 |

### 구조 요약
개념의 형성 과정을 설명한 뒤, 정의의 기능을 확정 작업으로 규정하는 구조

## 문장 04

> 하나의 체계는 여러 요소들이 일정한 원리에 따라 결합된 구조를 이루며, 개별 요소만으로는 설명할 수 없는 전체의 작동 방식이 존재한다.

### 문장 분해
- 정의: 체계 = 요소들이 원리에 따라 결합된 구조
- 주장: 개별 요소만으로는 전체 작동 방식을 설명할 수 없다

### 핵심 개념어

| 개념어 | 이 문장에서의 의미 |
|--------|---------------------|
| 체계 | 요소들이 규칙적으로 조직되어 하나의 전체를 이루는 짜임 |
| 요소 | 체계를 구성하는 개별 단위 |
| 원리 | 체계가 작동하는 근본적인 법칙이나 규칙 |
| 구조 | 요소들이 배치되고 연결되는 방식 |

### 구조 요약
체계의 구성을 정의한 뒤, 전체가 부분의 합 이상임을 암시하는 구조

## 문장 05

> 생물 분류 체계에서 '종'은 하위 개념이고 '계'는 상위 개념이며, 이러한 위계 관계는 범주 간의 포함 관계를 통해 전체 분류 체계를 형성한다.

### 문장 분해
- 예시: 종 = 하위 개념, 계 = 상위 개념
- 결론: 위계 관계가 범주 간 포함 관계를 통해 분류 체계를 구성한다

### 핵심 개념어

| 개념어 | 이 문장에서의 의미 |
|--------|---------------------|
| 하위 개념 | 더 좁은 범위를 가리키는, 상위에 포함되는 개념 |
| 상위 개념 | 하위 개념을 포괄하는 더 넓은 범위의 개념 |
| 위계 | 개념이나 요소가 상하 수준으로 배열되는 질서 |
| 범주 | 공통 기준에 따라 대상을 묶은 분류 단위 |
| 분류 체계 | 일정한 기준에 따라 대상을 위계적으로 나눈 전체 구조 |

### 구조 요약
구체적 예시로 상위/하위 개념을 설명한 뒤, 위계와 범주로 분류 체계를 구성하는 구조

## 문장 06

> 패러다임이란 특정 시기에 학문 공동체가 공유하는 이론적 틀과 기준을 뜻하며, 기존 패러다임으로 설명할 수 없는 현상이 누적되면 새로운 패러다임으로의 전환이 일어난다.

### 문장 분해

- 조건: 기존 패러다임으로 설명 불가능한 현상이 누적되면
- 결론: 새로운 패러다임으로 전환된다

### 핵심 개념어

| 개념어 | 이 문장에서의 의미 |
|--------|---------------------|
| 패러다임 | 특정 시대의 학문 공동체가 공유하는 이론·방법·가치의 총체적 틀 |
| 기준 | 판단이나 분류의 근거가 되는 잣대 |

### 구조 요약
패러다임을 정의한 뒤, 전환이 일어나는 조건을 설명하는 구조

## 문장 07

> 환원이란 복잡한 현상을 더 기본적인 단위로 분해하여 설명하는 방식이다. 그러나 의식이나 생명처럼 하위 단위로 쪼개면 본래의 성질이 사라지는 현상에 대해서는 환원불가능성이 제기된다.

### 문장 분해
- 정의: 환원 = 복잡한 현상을 기본 단위로 분해하여 설명
- 한계: 하위 단위로 쪼개면 성질이 사라지는 현상이 있다
- 결론: 이런 경우 환원불가능성이 제기된다

### 핵심 개념어

| 개념어 | 이 문장에서의 의미 |
|--------|---------------------|
| 환원 | 복잡한 현상을 더 단순한 구성 요소로 분해하여 설명하는 방법 |
| 환원불가능성 | 하위 요소로 분해하면 상위 수준의 고유한 성질을 설명할 수 없게 되는 한계 |

### 구조 요약
환원의 원리를 정의한 뒤, 환원불가능성이 제기되는 조건을 제시하는 구조

## 문장 08

> 창발이란 개별 요소에는 없던 새로운 성질이 요소들의 결합을 통해 전체 수준에서 나타나는 현상으로, 물 분자 하나에는 '젖음'이라는 속성이 없지만 다수가 모이면 젖음이 출현하는 것이 그 예이다.

### 문장 분해

- 예시: 물 분자 하나 → 젖음 없음, 다수 → 젖음 출현

### 핵심 개념어

| 개념어 | 이 문장에서의 의미 |
|--------|---------------------|
| 창발 | 부분에는 존재하지 않던 새로운 성질이 전체 수준에서 출현하는 현상 |

### 구조 요약
